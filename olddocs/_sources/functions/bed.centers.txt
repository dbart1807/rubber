###############
*bed.centers*
###############
calculate centers of genomic intervals

**************************************************************************
description
**************************************************************************

``bed.centers`` generates a bed file of 1-bp intervals corresponding to the central base each genomic interval.

**************************************************************************
usage
**************************************************************************

::

  bed.centers ( bedfile )


**************************************************************************
arguments
**************************************************************************

===========================      ===============================================================================================================================================================================================================
 Main options                     Description
===========================      ===============================================================================================================================================================================================================
**bedfile**                         a string defining the bed file name to find centers of
===========================      ===============================================================================================================================================================================================================


**************************************************************************
output
**************************************************************************
``bed.centers`` generates a bed file of 1-bp intervals corresponding to the central base of each genomic interval.


**************************************************************************
examples
**************************************************************************

make a bed of centers of MNase-seq fragments
"""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

.. highlight:: r

::

 > bed.centers ( "MNaseSeqFragments.bed" )
